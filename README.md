# City Building Game using Unity

## Prerequisites
- Download and Install [Unity](https://unity.com)
- Install Unity 2022 from Unity Hub
- Install an Editor of your choice - Visual Studio, VS Code or JetBrains Rider

## Get Started
- Open Terminal
- Clone this Repo `git clone https://github.com/KrishBaidya/CityBuilder-Unity.git`
- Import cloned Folder in Unity Hub
- Open Project
- Hit Play on the Editor
